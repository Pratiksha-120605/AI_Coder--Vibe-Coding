# Pratiksha Deshmukh — Portfolio (Next.js)

This is a minimal, professional Next.js site designed as an "Enterprise meets edgy" resume/portfolio.

Quick setup:

```bash
cd d:/Projects/SITE
npm install
npm run dev
```

Notes:

- Place `Pratiksha_Deshmukh_AI.pdf` in the `public/` folder so the resume link works.
- The site uses Tailwind CSS. If you prefer plain CSS, I can convert it.

Pages:

- `/` — Home
- `/about` — About me and resume download
- `/career` — Career timeline
- `/portfolio` — Placeholder for portfolio items

I'll iterate on visuals and copy if you'd like a bolder/softer tone, extra sections, or push a different layout.
